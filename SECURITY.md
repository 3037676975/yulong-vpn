# 安全说明

部署玉龙VPN 2.0 前，请先确认配置安全。

## 不要提交到 GitHub

```txt
.env
.env.local
真实后台登录信息
真实数据库密钥
真实业务节点信息
```

## 线上配置位置

线上配置请放到 Vercel：

```txt
Project Settings -> Environment Variables
```

## 后台登录

第一次登录使用环境变量里的后台邮箱和后台登录口令。登录后，可以在后台左侧菜单的账号密码设置中修改。

## 插件注意事项

浏览器插件只能访问公开接口，不能包含服务端敏感配置。

## 开源前检查

```txt
1. 检查仓库里没有 .env
2. 检查文档里没有真实登录信息
3. 检查插件里没有服务端敏感配置
4. 检查旧版本压缩包没有进入主线
5. 检查 fallback 节点是否可以公开
```
